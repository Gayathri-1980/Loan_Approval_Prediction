# Loan Approval Prediction using Machine Learning

This project predicts whether a loan application should be approved based on customer attributes using a supervised machine learning model.

---

## Project Summary :
The project includes:
- Data cleaning  
- Feature encoding  
- Model training  
- Performance evaluation  
- Prediction interface  

---

## Tech Stack :
- Python  
- Scikit-learn  
- Pandas  
- Matplotlib  
- NumPy
